# Slidev deck: Coding Agent × 将棋 AI

## Setup

```powershell
npm ci
```

## Live preview

```powershell
npm run dev
```

## Build (local)

```powershell
npm run build
```

## Build (GitHub Pages)

GitHub Actions では `npm run build:pages` を使って `--base "/<repo>/"` を自動設定します。

## Export PDF

```powershell
npm run export
```

## Assets

- 画像/動画などは `public/assets/` に置く（オフラインでも壊れにくい）
